Showcase

Conduit - Your event hub

This project leveraged the use of API's. The overall idea is to give users a place to shop for things to do and to save them and share them with friends. Currently, the site only taps in meetup.com; however, there are plans to expand and include more API's.

Technologies used

  • Language: JavaScript with Node.js
  • NPM packages: body-parser, connect-flash, cookie-parser, dotenv, ejs, express-ejs-layouts, express-session
  • Middleware: morgan, express,
  • Security: passport, passport-local, bcrypt-nodejs
  • Database: mongoose, mongodb hosted on heroku
  • Styling: HTML, CSS(Saas, Less), Bootsrap,
  • Site Hosting: Heroku
